# App-Tasks
Web application that allows users to create, edit, delete, and organize tasks. Created with Laravel for the backend and Vue.js for the frontend.

## Usage Vue.js

``` bash
$ cd front-end
$ npm install
$ npm run dev
```
## Usage Laravel

``` bash
$ cd App-Task
$ php artisan serve
```
